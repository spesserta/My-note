# 一、程序基本结构

```cpp

using namespace std;

// 主函数：程序入口，返回int类型
int main() {
    // 输出语句：cout是输出流，<<是流插入运算符
    cout << "Hello, C++!" << endl;
    
    // 返回0表示程序正常结束
    return 0;
}
```

# 四、流程控制
## 1. 条件语句

>if-else举例

```cpp

int score = 85;
if (score >= 90) {
    cout << "优秀" << endl;
} else if (score >= 80) {
    cout << "良好" << endl;
} else {
    cout << "继续努力" << endl;
}
```
>switch举例
```cpp

int day = 3;
switch (day) {
    case 1:
        cout << "周一" << endl;
        break;  // 跳出switch，否则继续执行后续case
    case 2:
        cout << "周二" << endl;
        break;
    default:  // 所有case不匹配时执行
        cout << "其他天" << endl;
}
```
## 2. 循环语句
>for 循环（适合已知循环次数）
```cpp
// 输出1-5
for (int i = 1; i <= 5; i++) {
    cout << i << " ";
}
```
>while 循环（适合未知循环次数）

```cpp

int num = 0;
while (num < 5) {
    cout << num << " ";
    num++;
}
```
>do-while 循环（至少执行一次）

```cpp

int a = 0;
do {
    cout << a << " ";
    a++;
} while (a < 5);
```
## 3. 循环控制
>break：跳出当前循环 /switch <br>
>continue：跳过本次循环剩余代码，进入下一次循环

# 五、数组
## 1. 一维数组
```cpp

// 定义方式1：指定长度并初始化
int arr[5] = {1, 2, 3, 4, 5};

// 定义方式2：省略长度，由初始化列表决定
int arr2[] = {10, 20, 30};

// 遍历数组
for (int i = 0; i < 3; i++) {
    cout << arr2[i] << " ";
}
```
## 2. 二维数组
```cpp

// 2行3列的数组
int matrix[2][3] = {{1,2,3}, {4,5,6}};

// 遍历二维数组
for (int i = 0; i < 2; i++) {
    for (int j = 0; j < 3; j++) {
        cout << matrix[i][j] << " ";
    }
    cout << endl;
}
```

# 六、函数
## 1. 函数基本结构
```cpp

// 函数声明（可选，若定义在main之后需声明）
int add(int a, int b);

// 函数定义：返回值类型 函数名(参数列表) { 函数体 }
int add(int a, int b) {
    return a + b;  // 返回计算结果
}

int main() {
    // 函数调用
    int result = add(3, 5);
    cout << "3+5=" << result << endl;  // 输出8
    return 0;
}
```
## 2. 函数重载
>同一作用域内，函数名相同、参数列表（个数 / 类型 / 顺序）不同：
```cpp

// 重载1：两个int相加
int add(int a, int b) {
    return a + b;
}

// 重载2：三个int相加
int add(int a, int b, int c) {
    return a + b + c;
}

// 重载3：两个double相加
double add(double a, double b) {
    return a + b;
}
```
# 七、指针（核心难点）
## 1. 指针基本概念
>指针：存储变量内存地址的变量 <br>
>语法：类型* 指针变量名;
```cpp

int num = 10;
int* p = &num;  // &取地址符：p存储num的地址

cout << "num的值：" << num << endl;    // 输出10
cout << "num的地址：" << &num << endl; // 输出内存地址
cout << "p的值：" << p << endl;        // 输出num的地址
cout << "p指向的值：" << *p << endl;   // *解引用符：输出10

*p = 20;  // 修改指针指向的变量值
cout << "修改后num的值：" << num << endl;  // 输出20
```
## 2. 指针与数组
>数组名本质是数组首元素的地址：
```cpp
int arr[] = {1,2,3};
int* p = arr;  // 等价p = &arr[0]

cout << *p << endl;    // 输出1（arr[0]）
cout << *(p+1) << endl;// 输出2（arr[1]）
```
