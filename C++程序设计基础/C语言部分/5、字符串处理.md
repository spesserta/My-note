# 一、字符串处理函数
>在使用字符串处理函数之前记得加上include<string.h>这个头文件。
## 1、strlen获取字符串长度

```c
#include <cstdio>
#include<cstring>
int main()
{
    char str[100] = "Hello";
    int size = sizeof(str);         //100个char类型占100字节
    printf("sizeof = %d\n",size);
    
    int len = strlen(str);         //Hello的长度为5
    printf("len = %d", len);
    return 0;
}
```

## 2、strcat字符串的拼接

```c
#include <cstdio>
#include<cstring>
int main()
{
    char str1[100] = "I LOVE ";
    char str2[4] = "YOU";
    strcat_s(str1,str2);  //将str2拼接到str1
    printf("%s",str1);
    return 0;
}
```

>请格外注意目标字符串后必须还有足够空间。如果目标字符串所在的数组仅有8个元素，这样拼 接后，多出来的字符将造成数组越界。

## 3、strcpy字符串复制函数

```c
#include <cstdio>
#include<cstring>
int main()
{
    char str1[100] = "I LOVE ";
    char str2[4] = "YOU";
    strcpy_s(str1,str2);  //将str2复制到str1
    printf("%s",str1);
    return 0;
}
```

## 4、strcmp字符串比较函数

>用于比较两个字符串，如果两个字符串一致，则返回0。更长的话返回正数、更短就返回负数。
```c
#include <cstdio>
#include<cstring>
int main()
{
    //开辟一片堆空间放字符串，并且指针指向他，注意这个内容不可修改，需要用const修饰
    const char* str1 = "abcde";  
    const char* str2 = "abcdef";
    const char* str3 = "ab";

    printf("%d\n", strcmp(str1, str2));   //表示str1比str2大还是小
    printf("%d\n", strcmp(str2, str3));   //表示str2比str3大还是小
    printf("%d\n", strcmp(str1, str3));   //表示str1比str3大还是小
    printf("%d\n", strcmp(str1, str1));   //俩相同大小的比较
    return 0;
}
```

<img width="391" height="232" alt="image" src="https://github.com/user-attachments/assets/5c99f1d2-3a71-494a-a8eb-19bfbb2c0e88" />


>注意他们之间比较的是当前字符的ASCII码,如果长度相同就比较ASCII码。

```c
#include <cstdio>
#include<cstring>
int main()
{
    //开辟一片堆空间放字符串，并且指针指向他，注意这个内容不可修改，需要用const修饰
    const char* str1 = "abcde";  
    const char* str2 = "bcdef";
    printf("%d\n", strcmp(str1, str2));   //表示str1比str2大还是小
    
    return 0;
}
```
