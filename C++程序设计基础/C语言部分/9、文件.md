# 一、多文件代码

## 1、预处理指令#include
>当编译器从上到下编译源文件 main.c 时，需要先知道标识符 printf 是一个函数。接着，在后面的代码中，我们才能将标识符 printf 当做函数来使用。而我们最常用的让编译器识别 printf 的方式就是使用预编译指#include<stdio.h>。

<br>

>预处理指令 #include ，会将文件 stdio.h 中的代码复制到该预处理指令出现处，并删除该预处理指令。

>修改后的代码将另存为中间文件或直接输入到编译器。并不会保存到源文件中。

## 3、多文件代码

>把函数的声明单独写在一个文件里面，谁需要使用这些函数，就包含这个文件就好。并且，这种文件不需要经过编译器编译，仅供被其他文件包含。具有这种性质的文件被称作头文件。区别于需要被编译器编译的文件，其后缀名用 .h 。

<img width="924" height="349" alt="image" src="https://github.com/user-attachments/assets/3a38aefd-d3fa-45d6-b7df-8d717886e06b" />

>新建一个头文件，由于它是源文件 print.c 对应的头文件。所以，这个头文件命名为 print.h 。
