# 一、内存地址
>众所周知，中央处理器CPU是由以下几个单元构成：
>- 算术、逻辑单元：对数据执行运算(例如加法、减法)的电路。
>- 控制单元：协调机器活动的电路。
>- 寄存器组 ：数据临时存储。
>除此之外还有内存负责数据的长期存储，内存和CPU之间通过数据总线相连接。

<img width="804" height="456" alt="image" src="https://github.com/user-attachments/assets/45fa3f3e-6a0a-4df5-ba68-39624d3feee9" />


>以2个数的加法为例演示运算过程：
>- 从内存中取出一个加数放到一个寄存器中。
>- 从内存取出另一个加数，放到另一个寄存器中。
>- 激活算术、逻辑单元中的加法电路，将前面两步的寄存器作为输入，用另一个寄存器存放结果。
>- 将结果存放到内存中。
>由此可以看出：只有正在被处理的数据才会被放到CPU的寄存器上，等待CPU进行计算，CPU将数据计算完毕后，再存 储到内存中。所以，内存才是数据的大本营。<br>
>8个晶体管构成一个字节，在内存中有许多字节，每个字节在内存中都有相应的编号，这个编号叫做内存地址：


<img width="817" height="432" alt="image" src="https://github.com/user-attachments/assets/c910c352-1db2-4ed9-94c8-c8eb997e182d" />


>不同的数据类型占用的字节也不同，有些数据类型需要占用多个内存。

<img width="805" height="349" alt="image" src="https://github.com/user-attachments/assets/0450efa4-5516-4137-b05a-e5f2ea7733ca" />


>因此要表示一个数据，除了要说出他的内存地址，也要说出它占用内存的大小。

<img width="820" height="218" alt="image" src="https://github.com/user-attachments/assets/17bf2c98-bf9e-43e7-a751-6c6c1237735d" />


>那么数据的第一个字节地址称为这个数据对象的首地址。那么数据占用的字节数量，就是它所占用的存储空间大小。
>这个数据的首地址叫做指针，这个占用空间大小由数据类型来决定。



# 二、指针数据类型
## 1、取地址运算符&
> 这个运算符写在一个数据对象的左边，可以获取一个数据对象的首地址和所需存储空间大小。

<img width="464" height="79" alt="image" src="https://github.com/user-attachments/assets/64be0394-83a4-401e-ad0e-8498f85d27a9" />


## 2、指针变量的声明

<img width="519" height="206" alt="image" src="https://github.com/user-attachments/assets/f994601c-e642-46da-95c4-d2b51e90fc4e" />


>例如：

```c
int n; int* pn = &n;     //int* pn 声明一个保存了 int 类型的首地址和大小的变量。 
char c; char* pc = &c;   //char* pc 声明一个保存了 char 类型的首地址和大小的变量。
```

>变量pn 存储了变量 n 的首地址与大小，变量 pc 存储了变量 c 的首地址与大小。通过 pn 和 pc 可以在内 存中找到变量 n 和 c 。<br>
>定义：设一个数据对象为 x ，设另一个数据对象为 p 。 p 存储了 x 的首地址和所占空间大小。那么， p 称之 为 x 的指针，或者说 p 指向 x 。

# 三、指针的运算

## 1、指针指向的前移后移

>对于int来说，int *p中的p+1，则指针指向的地址就+4字节，数组的话就是指向下一个变量的地址。p-1的话就是-4字节。<br>
>因此：<br>
>假设sizeof(目标数据对象) 被称作步长。 指针类型加 n 后。其首地址向后移动 n * 步长字节。 指针类型减 n 后。其首地址向前移动 n * 步长字节。<br>

## 2、修改指针指向的变量值

>使用取值运算符*，将指针指向的目标数据对象进行修改。<br>
>例如*p = 111 ，使得 p 指向的数据对象改为111。 *(p - 1) = 222 ，使得 p - 1 指向的数据对象改为222。 *(p - 2) = 333 ，使得 p - 2 指向的数据对象改为333。 *(p - 3) = 444 ，使得 p - 3 指向的数据对象改为444。<br>
>注意：需要将p的变化用括号括起来，因为运算优先性的缘故，*p-1就是先把p指向的值取出来然后把该值减一而不是先移动指针后取值。<br>
>还有要注意函数内声明的变量在内存中不一定是首尾相接的（数组可以），实际应用中不能这么写！<br>

## 3、已知指针首地址求变量占用的字节
>如果一个数组中int *p1=arr[0],*p2=arr[1]； 则p2-p1=n说明该单个数据变量的大小是n<br>


# 四、指针与数组

## 1、使用指针访问数组

>指针类型加n后。首地址向后移动 n * 步长 字节。 指针类型减n后。首地址向前移动 n * 步长 字节。因此，指针加减运算对于访问在内存中连续排布的数据对象非常方便。而数组正好满足此特点。

## 2、指针指向数组的首地址

>有2种方式：第一是直接指向数组的第一个元素，例如：int *p=&arr[0]; 还有一种方式就是直接使用数组的首地址，例如int *p=arr;因此数组名在C语言中可以直接表示第一个元素的首地址，但是有2个例外情况，第一个就是sizeof(arr)表示的是整个数组所占用的空间大小，第二种就是对数组名使用&运算符。

## 3、数组的本质

>数组的本质是指针。如下图可知使用指针访问数组等价于下标访问

```c
#include <cstdio>

int main()
{
    int arr[5] = { 1,2,3,4,5 };
    int* parr = arr;                 //指针指向数组首地址
    printf("arr[2] = %d \n",arr[2]);
    printf("*(arr + 2) = %d \n", *(parr+2));
    return 0;
}
```

# 六、多级指针

>如下图：int ** 是一个 int * 类型数据对象的指针，所以被称之为二级指针。和普通指针一样，它也是可以使用取值运算符*，来获取目标数据对象的。<br>
>取地址过程: 对n使用取地址运算符，获得n的指针pn，类型为 int * 。 对pn使用取地址运算符，获得pn的指针pnn，类型为 int ** 。<br>
>取值过程：对pnn使用取值运算符，将 int ** 还原为 int * 。对*pnn使用取值运算符，将 int * 还原为 int 。即，还原为n。<br>

```c
#include <cstdio>

int main()
{
    int n = 123;
    int* pn = &n;         //一级指针指向数据变量的地址
    int** pnn = &pn;      //二级指针指向一级指针的地址
    int*** pnnn = &pnn;   //三级指针指向二级指针的地址
    int**** pnnnn = &pnnn;//四级指针指向三级指针的地址

    printf("%d",****pnnnn); //四级指针这里输出原数据变量

    return 0;
}
```
