## 4、switch语句

<img width="576" height="440" alt="image" src="https://github.com/user-attachments/assets/dfdd5844-35f9-4ad3-9a50-18503c0d1a9b" />

>运算步骤：
>- switch 会先计算括号内的整型表达式结果。
>- 依次对比 case 中的常量，是否等于整型表达式的结果。
>- 若不相等，对比下一个 case 。 
>- 若相等，跳转到这个 case 。
>- 若没有相等的，跳转到 default 。

<img width="905" height="454" alt="image" src="https://github.com/user-attachments/assets/ab4dda02-e175-4b62-b52b-e8d96c863cd7" />

>注意：
>- switch 后面的括号里面只能填一个整型表达式。
>- case 后的常量不能有重复的。
>- 可以没有 default 。
>- break必须写上，否则后面的case都会执行，并且break并不能跳出switch语句

```c
switch (表达式) {
    case 常量1:
        // 代码块1
        break;
    case 常量2:
        // 代码块2
        break;
    ...
    default:
        // 默认代码块（可选）
}
```

# 三、循环结构

## 1、while循环

```c
while (条件表达式) {
    // 循环体（条件为真时执行）
}
```

>举例一个求和∑的例子

```c
#include <stdio.h>

int main() {
    int sum = 0;
    int num = 1;

    while (num <= 100) {
        sum += num; // 累加
        num++;     // 更新计数器
    }

    printf("1到100的和是：%d\n", sum);
    return 0;
}
```

## 2、do-while循环

```c
do {
    // 循环体（至少执行一次）
} while (条件表达式); // 注意结尾有分号

```


<img width="913" height="737" alt="image" src="https://github.com/user-attachments/assets/e050810a-f06c-4109-9aec-c89a0890498e" />



>相比于while，它至少能执行一次，而且是先执行后判断，举例如下：计算用户输入数字的平方

```c
#include <stdio.h>

int main() {
    int num;

    do {
        printf("请输入一个数字（输入0退出）：");
        scanf("%d", &num);
        if (num != 0) {
            printf("平方是：%d\n", num * num);
        }
    } while (num != 0); // 当输入不是0时继续循环

    printf("程序结束。\n");
    return 0;
}
```

## 3、for循环

```c

for (初始化; 条件; 更新) {
    // 循环体（条件为真时执行）
}

```

>举例1到100的偶数和

```c
#include <stdio.h>

int main() {
    int sum = 0;

    for (int i = 2; i <= 100; i += 2) { // 从2开始，每次+2
        sum += i;
    }

    printf("1到100的偶数和是：%d\n", sum);
    return 0;
}
```
