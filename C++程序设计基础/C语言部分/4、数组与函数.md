# 三、字符串和字符数组
## 1、字符串常量：

<img width="643" height="249" alt="image" src="https://github.com/user-attachments/assets/6e24c0f9-4154-45d3-9550-f7bd3a2d69fb" />


> 字符串常量 "HelloWorld" 占用11个字节。字符串常量在内存中，由每个字符的ASCII码按照顺序排列构成，每个字符仅占一个字节，并且末尾会 附上一个数值0，指示字符串结尾。字符'0'对应的ASCII码为十进制48。而标记字符串结尾的数值0，为了不与字符'0'冲突，使用转义序 列'\0'表示。

## 2、字符数组存储字符串
>结尾不加/0的初始化：char str[20] = {'H', 'e', 'l', 'l', 'o', 'W', 'o', 'r', 'l', 'd'}; <br>
>结尾加/0的初始化：char str[20] = "HelloWorld"; 等价于char str[20] = {'H', 'e', 'l', 'l', 'o', 'W', 'o', 'r', 'l', 'd', '\0'};<br>

3、函数声明
>如果函数没有声明并且函数在主函数后边，主函数就会找不到被调函数就会报错，这时候就需要声明一下<br>

4、类型转换
形参与实参的数据类型不一致，调用函数时会转成形参的数据类型，实参数据类型更大的话可能丢失精度。<br>
5、形参与实参相互独立
虽然主函数中的变量a，b与函数中的形式参数a，b变量名相同。但是，它们却是相互独立的变量。<br>
函数内声明的变量为局部变量，不同函数内的局部变量相互独立。<br>
6、函数的递归
函数内部是可以再次调用自己的。这种调用被称之为 函数递归 ，递归是以函数栈的形式产生，满足栈先调用后删除的特点。递归的好处是可读性强，但是坏处就是时间复杂度极高，不能适用于数据量大的情况。当函数递归到最顶层时会回溯，如下图红线表示递推，蓝线表示回归
