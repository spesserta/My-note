# 二、Hello world

>无需多言

```C#
    using System;
    class Program      //该cs文件的主类
    {
        static void Main(string[] args)    //主函数
        {
            System.Console.WriteLine("Hello world!");  //输出一方Hello world并且换行
            System.Console.Write("Hello world!");      //输出Hello world不换行
            //因为VS默认引用了System命名空间，所以前面的System.可以不用写
            Console.WriteLine("Hello World!");
            Console.Write("Hello world!");  
        
        }
    }

```

```c#
   using System;

       {
           //有符号整数

           //自动类型变量，会根据变量是什么自动匹配类型
           var i = 1.2;  //double
           var b = 100;  //int
           var c1 = false; //bool
           var c2 = 218.217F; //float
       }
   }

```
## 3、变量的输出

```C#
    using System;
    class Program      //该cs文件的主类
    {
        static void Main(string[] args)    //主函数
        {
        int a = 3;
        int b = 4;
        Console.WriteLine(a + b);
        Console.WriteLine();
        Console.WriteLine("a+b=result");
        Console.WriteLine();
        int result = a + b;
        Console.WriteLine("{0}+{1}={2}",a,b,result); //大括号括起来的数字表示先后顺序（必须从0开始），这种输出方式类似于C语言的占位符%d
        Console.WriteLine($"{a}+{b}={result}");   //加了美元符号$后可以在大括号里直接写出需要输出的变量
        } 
    }


```

## 5、变量的类型转换

>根据计算机组成原理，每种变量的字长不一样，类型转换有相同字长的转换、大字转小字、小字转大字这3种情况。<br>
>其中相同字长的转换不会改变内存中的数据，只是计算机对这个内存中的数据的解释会不一样，例如int和uint。<br>
>大字转小字，如果大字数值范围过大，超出了小字的范围则会丢失精度，例如double转float<br>
>小字转大字，不丢失精度，例如float转double<br>
>浮点型转整型，小数部分会被砍掉<br>


<img width="482" height="335" alt="image" src="https://github.com/user-attachments/assets/f6a97d41-2e05-470a-9393-fa7f646bb005" />

>里面的(int)d表示强制类型转换，可能会发生风险，用这个相当于告诉编译器可以无视转化的风险。<br>
>与此对应的还有隐式类型转换，这个是编译器自动发生的，没有风险，例如float变量加上double变量，会先把float转换成double再进行运算。<br>
