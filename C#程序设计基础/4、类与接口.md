# 一、类与封装

## 1、类的数据成员

>一个类包括其数据成员和成员函数，声明一个类的话，class默认是public权限，数据成员默认是private权限

```C#

namespace ConsoleApp3
{
    class Person  //声明一个人类（是一个抽象概念不是具体的某个人）
    {
        //public表示公共权限，类外的方法可以直接通过打点访问
        public string Name; //人类的名字
        public int age;     //人类的年龄
        //private表示私有权限，类外方法无法直接访问到，只能通过本类的成员函数作为接口访问
        private int Id;     //人类的身份证号
        //protected表示保护权限，在继承的时候权限会有区别
        protected bool sex;  //人类的性别

        //类中的数据成员如果用static静态修饰，那么就表示全体该类所有对象共有的属性
        public static int PersonMaxAge = 150; //人类的最大年龄无法超过150
        public static int PersonLong = 300;   //人类的最高身高不超过300cm
    }
    internal class Program
    {
        static void Main(string[] args)
        {
            //申明类的对象
            Person person1=new Person(); //new一个Person类的对象
            Person person2=new();        //new后面可以把重复的Person类型去掉，这只是个语法糖
            //给类的对象赋值
            person1.Name = "小明";
            person1.age = 20;
            person1.Name = "小红";
            person1.age = 12;
            person1.Id = 1212;   //private权限外面无法打点访问
            person1.sex = false; //protected权限外面无法直接访问
            //访问类的对象
            Console.WriteLine($"姓名：{person1.Name} 年龄：{person1.age}");
        }
    }
}

```

## 2、类的成员函数

>C#不同于C语言的全局函数，C#的函数必须是在某个类里面做成员方法，需要先搞一个类然后成员函数规定这个类可以干什么

<img width="682" height="176" alt="image" src="https://github.com/user-attachments/assets/20966a56-3b8d-44d8-bacd-2fbd025f3578" />

>把它放在类里就可以了，注意Main 是 static 方法，而 static 方法不能直接调用非 static 方法

<img width="596" height="384" alt="image" src="https://github.com/user-attachments/assets/6b967e69-8676-4016-8517-2f1076d3104f" />

>如果非要调用的话，可以将成员函数改成static静态类型。

```C#
    internal class Program
    {
        static void Awa()
        {
            Console.WriteLine("awa");
        }
        static void Main(string[] args)
        {
            Awa();
        }
    }
```

>或者说可以先声明一个具体的实例，然后通过具体的实例来调用
```C#
    internal class Program
    {
        void Awa()
        {
            Console.WriteLine("awa");
        }
        static void Main(string[] args)
        {
            Program program = new Program();
            program.Awa();
        }
    }
```

>成员函数还可以声明成带返回值的

```C#
    internal class Program
    {
        int Awa()
        {
            Console.WriteLine("awa");
            return 0;
        }
        static void Main(string[] args)
        {
            Program program = new Program();
            int a=program.Awa();
            Console.WriteLine($"{a}");
        }
    }

```

<img width="386" height="110" alt="image" src="https://github.com/user-attachments/assets/15900d62-96b7-4316-8cce-996c361a2949" />


>当然这种简单的成员函数可以用语法糖的形式来写出来：

```C#
    internal class Program
    {
        int Awa() => 0;   //直接省略return和括号让函数更加简单易懂
        
        static void Main(string[] args)
        {
            Program program = new Program();
            int a=program.Awa();
            Console.WriteLine($"{a}");
        }
    }

```

>一个人类的完整定义如下：

```C#
namespace ConsoleApp3
{
    class Person  //声明一个人类（是一个抽象概念不是具体的某个人）
    {
        //public表示公共权限，类外的方法可以直接通过打点访问
        public string Name; //人类的名字
        public int age;     //人类的年龄
        //private表示私有权限，类外方法无法直接访问到，只能通过本类的成员函数作为接口访问
        private int Id;     //人类的身份证号
        //protected表示保护权限，在继承的时候权限会有区别
        protected bool sex;  //人类的性别

        //类中的数据成员如果用static静态修饰，那么就表示全体该类所有对象共有的属性
        public static int PersonMaxAge = 150; //人类的最大年龄无法超过150
        public static int PersonLong = 300;   //人类的最高身高不超过300cm

        public void FrontForward()
        {
            System.Console.WriteLine("向前移动");
        }
        public void BackOutward()
        {
            System.Console.WriteLine("向后移动");
        }
        public void DownForward()
        {
            System.Console.WriteLine("向下移动");
        }
        public void UpForward()
        {
            System.Console.WriteLine("向上移动");
        }
    }
    internal class Program
    {
        int Awa() => 0;   //直接省略return和括号让函数更加简单易懂
        
        static void Main(string[] args)
        {
            Person person = new Person();
            person.FrontForward();
            person.BackOutward();
            person.DownForward();
            person.UpForward();
        }
    }
}
```
