# 异常处理

>异常就是程序运行的时候发生错误或者预期之外的情况，例如数组越界、空引用、文件不存在、参数错误等。在C#中可以使用try/catch/finally来处理异常

```c#
namespace ConsoleApp4
{
    internal class Program
    {
        static void Main(string[] args)
        {
            int[] arr = { 1, 2, 3 };
            System.Console.WriteLine(arr[3]);   //数组越界

        }
    }
}
```

>运行后控制台就出现了异常的报错，如果异常不及时处理的话可能会造成程序的崩溃

<img width="1321" height="449" alt="image" src="https://github.com/user-attachments/assets/9638a257-d534-4fca-87ce-e0143c5b0ccf" />

>可以使用try-catch来解决，原理就是try表示捕获异常的范围，catch表示异常的各种情况，括号里用异常对应的处理语句

```c#
namespace ConsoleApp4
{
    internal class Program
    {
        static void Main(string[] args)
        {

            try
            {
                int[] arr = { 1, 2, 3 };
                System.Console.WriteLine(arr[3]);   //数组越界
            }
            catch (IndexOutOfRangeException)  //异常情况1
            {
                Console.WriteLine("IndexOutOfRangeException");

            }
            catch (Exception)  //这个是异常的兜底，只要是个异常且前面的catch没有查到就执行这段
            {
                Console.WriteLine("Exception");
            }
        }
    }
}


```

>异常这东西肯定是希望能捕获到更精确的异常，所以不建议只用一个兜底的来捕获。
