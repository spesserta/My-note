# 一、线性表的定义和基本操作

## 1、线性表的定义：
  
>- 线性表是具有相同数据类型的n个数据元素的有限序列，其中n为表长，n=0时候线性表是空表。<br>
>- 如果用L命名线性表，这一般表示为：L=(a1,a2,a3,a4...ai)。其中a1为表头元素，ai为表尾元素，i为位序，从1开始，下标从0开始。除了第一个元素，每个元素有且仅有一个直接前驱。除了最后一个元素每个元素有且仅有一个直接后继。

# 二、顺序表的定义
> 顺序表是用顺序存储的方式实现线性表。顺序存储是把逻辑上相邻的元素存储在物理位置上也相邻的存储单元中，元素之间的关系由存储单元的邻接关系来体现。

## 1、顺序表结构体定义：

```cpp
   #define MaxSize 10 
   typedef struct{
        ElemType data[MaxSize];   //数据元素
        int length;                           //表长
   }Sqlist;
```

## 2、初始化一个顺序表：

```cpp
 void InList(Sqlist &L){
    for(int i=0;i<MaxSize;i++)   //给各个元素赋初始值
       L.data[i]=0;
       L.length=0;                        //表长初始值为0
   }
```

## 3、动态可增加长度的线性表实现：

```cpp
  #inlcude<stdlib.h>
  #include<stdio.h>
  #define InitSize 10 //默认最大长度
  typedef struct{
      int *data;      //动态分配数组的指针
      int MaxSize; //顺序表的最大容量
      int length;    //顺序表的最大长度
  }SeqList;

  void InitList(SeqList &L){
      L.data=(int *)malloc(InitSize*sizeof(int));//data指针指向新申请的一块内存区域
      L.length=0;   //初始化当前长度
      L.MaxSize=InitSize; //初始化最大长度
  } 

  void IncreaseSize(Seqlist &L,int len){
      int *p=L.data;      //p指针用于释放原来空间的内存
      L.data=(int *)malloc((L.MaxSize+len)*sizeof(int)); //扩大data指针指向的空间大小
      for(int i=0;i<L.length;i++)
            L.data[i]=p[i];   //将数据复制到新区域
       L.MaxSize=L.MaxSize+len;     //最大长度增加len个
       free(p);  //释放p指针指向的内存
  }
```

>缺点：花费一些时间

## 4、顺序表的特点：
>- 随机访问：已知是第几个元素，访问的时间复杂度是O(1)
>- 存储密度高：每个节点只存储数据元素，节省空间
>- 扩展容量不方便。
>- 插入删除需要移动大量元素，不方便。


# 三、顺序表的操作

## 1、插入操作ListInsert(&L,i,e):

```cpp
  bool ListInsert(SqList &L,int i,int e){
       if(i<1||i>L.length+1)   return false;    //插入位置不合法
       if(L.length>=MaxSize)  return false;  //满了
       for(int j=L.length;j>=i;j--)                  //插入前得腾出位置
             L.data[j]=L.data[j-1];                    //从插入位置开始到最后一个元素位置，所有元素向后移动一位
       L.data[i-1]=e;                                     //插入元素e在指定位置
       L.length++;                                        //表长+1
       return true;
}
```
>时间复杂度O(n)

## 2、删除操作ListDelete(SqList &L,int i,int &e)：

```cpp
    bool ListInsert(SqList &L,int i,int &e){
       if(i<0||i>L.length)   return false;       //删除位置不合法
       if(L.length==0)  return false;           //没有元素了
       e=L.data[i-1];                                   //备份数据
       for(int j=i;j<L.length;j++)
            L.data[j-1]=L.data[j];                   //从删除位置的后一位开始到最后一个位置，依次覆盖前驱元素
        L.length--;                                       //表长-1
        return true;
}
```
>时间复杂度：O(n)

# 五、单链表的插入和删除

## 1、按照位序插入ListInsert(&L,i,e)(带头结点):
```cpp
    bool  ListInsert(LinkList &L,int i,ElemType e){
       if(i<1)   return false;                                          //插入位置不合法
       LNode *p=L;                                                     //p指向链表的头结点
       int j=0;                                                              //当前的p指向第0个节点（头节点）
       while(p!=NULL&&j<i-1)                                  //找到插入位置的前驱结点
       {
           p=p->next;                                                  //指针后移
           j++;
        }
        if(p==NULL)   return false;                              //i不合法
        LNode *s=(LNode *)malloc(sizeof(LNode));   //创建一个新节点，s指向他
        s->data=e;                                                      //新节点data域赋值
        s->next=p->next;                                           //开始插入
        p->next=s;
        return true;
      }
```
>时间复杂度O(n)

## 2、按照位序插入ListInsert(&L,i,e)(不带头结点):

```cpp
//基本与上边相同，但是需要考虑i=1时候头结点更新的情况
    bool ListInsert(LinkList &L,int i,ElemType e){
           if(i<1)      return false;
           if(i==1){
          LNode *s=(LNode*)malloc(sizeof(LNode));
          s->data=e;   
          s->next=L;                                                                     //s接在头结点的前面
          L=s;                                                                                //把s变成头结点
          return true;   }
          LNode *p=L;                                                             //p指向链表的头结点
          int j=1;                                                                     //当前的p已经指向了第1个节点
          while(p!=NULL&&j<i-1)                                       //p最终指向插入位置的前驱结点
         {
            p=p->next;                                                               //指针后移
            j++;
         }
          if(p==NULL)   return false;                                     //没找到对应地址
          LNode *s=(LNode *)malloc(sizeof(LNode));  //创建一个新节点，s指向他
          s->data=e;                                                                    //新节点data域赋值
          s->next=p->next;  
          p->next=s;
          return true;
          }
```
>平均时间复杂度O(n)

## 3、指定节点的后插操作：

## 4、指定节点的前插操作：
```cpp

        LNode *s=(LNode *)malloc(sizeof(LNode));
        if(s==NULL)   return false;
        s->next=p->next;                                              //后插
        p->next=s;
        s->data=p->data;                                             //把p的数据移动到后边的s的data域

                s->next=p->next;                                       //后插
                p->next=s;

}
```

## 5、按位序删除ListDelete(&L,i,&e)（带头结点）：

```cpp
   bool ListDelete(LinkList &L,int i,ElemType &e){
    if(i<1)    return false;
    LNode *p=L;                                                              //p用于扫描满足条件的节点
    int j=0;                                                                      //p指向第0个节点
    while(p!=NULL&&j<i-1)                                          //找到要删除节点的前驱节点
     {
            p=p->next;                                                      //指针后移
            j++;
      }
     if(p==NULL) return false;                                       //没找到
     if(p->next==NULL)   return false;                          //前驱结点后没有节点了
     LNode *q=p->next;                                               //q用于指向要删除节点，辅助删除
     p->next=q->next;
     free(q);
     return true;
}
```
>平均时间复杂度O(n)

## 6、指定节点的删除DeleteNode(p)
```cpp
   bool DeleteNode(LNode*p){
       if(p==NULL) return false;   
        LNode *q=p->next;                                           //令q指向p的后继节点
       p->data=p->next->data;                                   //和后继节点交换数据域
       p->next=q->next;                                              //将q节点断开
       free(q);                                                               //释放q节点内存
      return true;
   }
```
>平均时间复杂度O(1)

# 七、单链表的建立

## 1、尾插法建立
```cpp
  LinkList list_TailInsert(LinkList &L){
     char x;
     L=(LinkList)malloc(sizeof(LNode));       //建立一个头结点
     LNode *s,*r=L;                                      //r为表尾指针
     x=getchar();                                          //输入插入节点的data值,回车结束
     while(x!='\n'){
           s=(LNode *)malloc(sizeof(LNode)); 
           s->data=x;            
           r->next=s;                                      //把新节点接在尾结点后面
           r=s;                                                 //更新尾结点
           x=getchar();
              }
       r->next=NULL;                                    //尾结点的next为NULL
       return L;
}
```
>平均时间复杂度O(n)

## 2、头插法建立
```cpp
LinkList list_HeadInsert(LinkList &L){
       LNode *s;
       char x;
       L=(LinkList)malloc(sizeof(LNode));  //创建一个头指针
       L->next=NULL;                               //初始化头结点为空链表（防止野指针）
       x=getchar();
       while(x!='\n')
       {
           s=(LNode*)malloc(sizeof(LNode));
           s->data=x;
           s->next=L->next;                         //头插
           L->next=s;
           x=getchar();
       }
       return L;
}
```
>平均时间复杂度O(n)。
>头插法还能将一个原来的单链表逆置，直接断开加头插就行

```cpp
    if(p==NULL)  return false;

    if(q==NULL)   return false;
    p->next=q->next;                                          //删除（共2句）
    if(q->next!=NULL)
         q->next->prior=p;
    free(q);
  return true;
}
```

# 九、循环链表
## 1、定义：相比于非循环链表，循环链表的表尾节点的next域指向头结点，这样可以增加链表的灵活性。单链表从一个节点出发就只能找到后续的各个节点，而循环链表从一个节点出发可以找到任何一个节点。如果对链表的操作都是在头部和尾部，那么可以让指针指向表尾元素，方便修改。

# 十一、顺序表和链表的对比

>- 逻辑结构上：都是线性结构，线性表

>- 存储结构上：顺序表存储密度高，支持随机存取，但是大片连续空间分配不方便，改变容量不方便。链表的离散的小空间分配方便，改变容量方便。但是不可以随机存取，存储密度低

>- 创建节点上：顺序表需要预留大片的连续空间，空间小了不够用，大了浪费内存.链表只需要分配一个头结点即可

>- 增删节点上：顺序表插入删除需要频繁移动数据元素。 O(n)。链表找到目的位置直接插入删除即可，更快一点。 O(n)

>- 查找节点上：顺序表按位查找可以直接找到(O(1))，按值查找O(n)。链表按值查找和按位查找都为O(n)
