# 欢迎来到计算机科学的世界！

我现在已经大四了，这是我大学四年的学习笔记汇总，如果你对计算机有兴趣，希望这个笔记能帮到你。💓 <br>

各科建议学习顺序：<br>

1、《C语言程序设计》  万恶之源，谁的大学生活不是从用C语言输出Hello World开始的呢？如果学完C语言后想深入此领域，并且对硬件有兴趣，可以了解下“嵌入式单片机”这个方向哦awa<br><br>
2、《C++程序设计》  这TM简直是牢完了，初学C++容易会有C++很简单的错觉，那是因为你学的不够深入，在C++中你会第一次接触到面向对象编程，等OOP的封装继承多态学完，还有重载、虚函数、内存管理、智能指针、STL容器等等，学习难度直线上升，到后期直接淹没在各种恶心的语法中，狗都学不会啊，我止步于STL，接下来的路靠你们走了。<br>
>编程语言到此为止吧，除了C/C++还可以了解C#和Java哦~，接下来是计算机专业课牢九门，一门更比一门牢



3、牢大《数据结构与算法》 孩子们不开玩笑，这门课非常重要，无论是面试考研，都离不开这门课，前期顺序表链表轻松拿捏，中期树图略显乏力，后期红黑树B树难如登天，一定要好好学，什么差都不能差数据结构啊！！<br>
4、牢二《计算机组成原理》 这位在难度上更是重量级，如果以后只想做个纯敲代码的，这门课及格就行了，但是及格也一点不容易啊，此功若成，定能原地飞升，羽化而登仙。<br>
5、牢三《操作系统》 个人认为最枯燥的一门，各种进程之间的博弈，饥饿死锁学到头秃，学完秒变管理大师，进程调度这章是重点中的重点，希望能把握住。<br>
6、牢四《计算机网络》 计算机之间是怎么通信的呢？这本书能解决你的疑问，下至物理层的接光纤，上至应用层的HTTP，直接学通，此书知识点过于零散，建议把握住重点来学w。<br>
